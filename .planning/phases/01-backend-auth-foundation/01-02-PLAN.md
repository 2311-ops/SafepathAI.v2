## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| App process -> device secure storage | Access/refresh tokens at rest on the device |
| App -> backend API | Tokens transit over the network on every authenticated call |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-02-01 | Information Disclosure | Auth tokens at rest on device | high | mitigate | Tokens stored exclusively in `flutter_secure_storage` (Keystore/Keychain-backed); no `shared_preferences`; asserted by the "no shared_preferences import" acceptance check (Task 2) |
| T-02-02 | Information Disclosure | Tokens/PII in logs | medium | mitigate | dio interceptor does not log bearer tokens or request bodies; refresh token never printed |
| T-02-03 | Tampering | Man-in-the-middle on API calls | medium | mitigate | Base URL is HTTPS in non-dev builds; `--dart-define=API_BASE_URL` supplies the real host (localhost only for dev) |
| T-02-SC | Tampering | pub.dev package supply chain | medium | mitigate | All Dart packages VERIFIED + Approved in RESEARCH.md Package Legitimacy Audit (no ASSUMED/SUS/SLOP); versions pinned in pubspec.yaml |
